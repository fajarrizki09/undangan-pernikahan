# Project Map

- `frontend/`: static public invitation and admin panel.
- `frontend/js/public/`: public page helper runtimes.
- `frontend/js/admin/`: admin panel helper modules.
- `frontend/js/shared/`: normalization, catalogs, Drive helpers, and shared schemas.
- `api/`: Vercel serverless handlers.
- `backend-gas/Code.gs`: Google Apps Script API for Sheets and Drive.
- `tools/ai-snapshot.ps1`: compact context snapshot for future AI sessions.
